# Students Analyzing Dashboard

## Project Overview
This project is an end-to-end student analytics dashboard developed using SQL, Excel, Power BI, and Python.

The dashboard analyzes:
- Student enrollments
- Academic performance
- Fee management
- Placement tracking

## Tools Used
- SQL
- Microsoft Excel
- Power BI
- Python

## Files Included
- SQL query files
- Power BI dashboards
- Excel datasets
- Python dashboard scripts

## Key Features
- Student performance analysis
- Fee tracking dashboard
- Placement analytics
- Course enrollment insights
